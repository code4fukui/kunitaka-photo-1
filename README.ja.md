# kunitaka-photo-1

このリポジトリには、拡張現実（AR）に適した高校の3Dアセットと、動画処理のサンプルコマンドが含まれています。

## 拡張現実用の3Dアセット

本リポジトリのメインとなるのは、AR開発ですぐに利用できるフォーマットで提供される、高校の3Dスキャンデータです。

### ARプレビュー

以下のリンクを開くと、iOSまたはiPadOSデバイスを使用して、メインの3Dモデルを拡張現実（AR）でプレビューできます。これにより、実際の環境に校舎を配置して確認することができます。

- [高校モデルをARで表示](video/h18.usdz)

### アセットファイル

3Dモデルは以下のフォーマットで提供されています:
- **`.usdz`**: AppleプラットフォームのARKitでネイティブに利用できます。主要なファイルとして `kunitaka-highschool.usdz` があるほか、`threed/` ディレクトリにはScaniverseアプリでスキャンした追加データが含まれています。
- **`.glb`**: 3Dシーンの標準フォーマットです。`kunitaka-highschool.glb` を参照してください。

## 動画処理ユーティリティ

このリポジトリには、動画処理用のコマンドラインスニペットのサンプルも含まれています。

### 動画の分割

FFmpegを使用して動画を60秒ごとのセグメントに分割するには:

```sh
ffmpeg -i IMG_6275.MOV -map 0 -c copy -f segment -segment_time 60 -reset_timestamps 1 IMG_6275_%02d.MOV
```

### 動画から画像への変換

動画を連続したJPEG画像に変換するには（`video2jpg` スクリプトにパスが通っていることを前提とします）:

```sh
video2jpg
```
